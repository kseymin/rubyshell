# 사용법

1.먼저 현재 레포지토리를 클론해온다.

<pre><code>$ git clone https://github.com/kseymin/rubyshell</code></pre>

2.스트립트에 실행 권한을 주자

<pre><code>$ $ sudo chmod u+x ~/rubyshell/shell1.sh</code></pre>

3.실행하자
<pre><code>~/rubyshell/shell1.sh</code></pre>

4. 중간에 루비버전을 물어본다 쓰자
예시 "input Ruby Version" <= 2.4.0
